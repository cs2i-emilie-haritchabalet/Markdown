# Titres
Pour structurer vos notes, vous pouvez leur donner des titres. Le signe # créé un titre.
=== " :material-file-document: Exemple"

    ```
    # H1 Header
    ## H2 Header
    ### H3 Header
    #### H4 Header
    ##### H5 Header
    ###### H6 Header

    Ceci est un paragraphe simple
    ```
